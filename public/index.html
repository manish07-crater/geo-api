Documentation

Developer Reference

Integrate India's village data into your apps in minutes.

1. Authentication

All API requests must include your API Key in the x-api-key header.

curl -H "x-api-key: YOUR_API_KEY" https://api.geopin.in/api/search?q=Amod

2. Search Village

GET /api/search

Search through 600,000+ villages by name, district, or sub-district.

PARAMETERS

q The search query (string) Required
limit Number of results (1-50) Optional (Default: 10)

RESPONSE EXAMPLE

{
  "success": true,
  "data": [
    {
      "village": "Amod",
      "sub_district": "Amod",
      "district": "Bharuch",
      "state": "Gujarat",
      "code": "522198"
    }
  ]
}
